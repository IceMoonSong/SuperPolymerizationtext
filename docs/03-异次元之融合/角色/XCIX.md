# XCIX

是泰伦文明的科技宗教月光会实际信奉的神明，对外被尊称为“织星者，创造之基，律动的星河，原粹之神，至圣之天球”；

月光会称呼其为一名有着八字辫的白玉女子。实际外形是一颗狼首状黑洞。

## 认知？

祂认为：宇宙应当维持自持性秩序。秩序与效率高于一切。“混沌”不是恶，除非干扰到秩序。

## 与月光会

月光会创立者乔伊·弗罗斯特在知道XCIX后，为利用教团网络搜寻神之键，故意将其包装为“创世神”，只对核心层传递真相。经数代传递后，连多数高级执事都分不清哪个是真正的教义。因此森口智子后来自称“救世主XCIX的代言人”——他并非背叛月光会，只是回归了其被尘封的真相。

## 真实身份

XCIX是宇宙中自然涌现的清除程序，旨在清除那些可能颠覆宇宙秩序的过度发育文明。其存在背景可追溯至数万亿年前：彼时高等文明因争夺资源爆发维度战争，每过一个生命时代，就有降维武器将原本 11 维的宇宙削减一个维度。为阻止文明继续破坏宇宙结构，宇宙的自持性逻辑催生了此类清理程序。

在艾俄斯神话《沐经》与《沐经注疏》中，XCIX被称为 “清理者”，是虚数之树的使者。很长一段时间内被视为虚构传说，直至艾俄斯文明覆灭时才被证实。
