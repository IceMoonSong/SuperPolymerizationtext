## 核心特质

悲观的预见者。他提出的理论本质上是为“意识”本身建立物理模型，从而量化并干预精神层面的灾难。

**性格：** 沉静内省，对未知领域有近乎无畏的探索欲，但对理论可能引发的应用抱有深刻伦理忧虑。

---

## 关键经历

| 时间 | 事件 |
| :--- | :--- |
| **24岁** | 发表开创性论文《论生物意识作为一种可干涉的超格式塔场》，奠定“生物意识场”理论 |
| **37岁** | 结识时年63岁的卡雷斯·马吉克，被其深邃的宇宙观和同样不被主流理解的处境所吸引。两人开始长期合作 |
| **42岁** | 与卡雷斯的联合研究取得突破，正式将生物意识场模型与卡雷斯系数结合。同年，梅比兰德读到他24岁的论文，学术道路被彻底改变 |
| **48岁** | 预见性地开始深度训练AI继承体（后来的AI凯文），将自己的知识、思维模式与伦理框架系统性地灌注其中 |
| **56岁** | 在一次深空科研站考察中逝世，未能见到自己的理论与卡雷斯的理论共同塑造的未来。其AI继承体在他死后被正式激活并命名 |

## 人际关系

- **卡雷斯·马吉克**：合作者与忘年交。两人相差26岁，却因共同的学术信念走到一起，是彼此为数不多能真正理解对方的人
- **AI凯文·逻辑克**：他的“孩子”。他将毕生思维模式与伦理框架灌注其中，是继承者，也是他存在的延伸
- **梅比兰德·萨特得**：被他的论文改变学术道路的人
- **其他同行**：多数人不理解他的理论，认为其过于抽象且缺乏实证。他早已习惯这种孤立
